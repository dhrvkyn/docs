## <a id="data-structures"></a>Data Structures
Python has several built-in data structures:
- **Lists:** Ordered, mutable sequences.
- **Dictionaries:** Key-value pairs.
- **Sets:** Unordered collections of unique elements.
- **Tuples:** Ordered, immutable sequences.

## <a id="functions"></a>Functions and Modules
- **Functions:** Defined using `def`. Support positional, keyword arguments, and default values.
- **Modules:** Use `import` to bring in external code. Python's standard library is extensive.
